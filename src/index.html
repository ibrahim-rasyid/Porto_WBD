Projects

  1. Biometric Fingerprint Matching Desktop App

    This program implements a biometric identification system using fingerprints, utilizing the Boyer-Moore and Knuth-Morris-Pratt methods for detection and linking fingerprints to a database for complete identification. The Regex algorithm is also used to manage potential data corruption during the matching process.

  2. WikiRace Website

    WikiRace is a game where players navigate from one Wikipedia article to another in the shortest time or fewest clicks, with the program implementing Iterative Deepening Search (IDS) and Breadth- First Search (BFS). The program outputs the number of articles examined, depth, exploration route, search time, and a directed graph visualization of the route.

  3. Two Player GUI-Based Field Management Game

    The application "Walikota! : Simulasi Kota Solo" is a two-player GUI-based game developed in Java, where each player starts with a deck of 40 cards and can hold a maximum of 6 cards in their active deck. Players cultivate a 4x5 field by planting crops or raising animals, and the winner is determined after 20 turns based on the amount of money accumulated.

  4. Word Ladder Solver

    The UCS, Greedy Best First Search, and A* algorithms are three search algorithms used to find a target node from a specific starting node in a graph. In this program, we implement these three algorithms to solve the word ladder game and analyze their optimization.

  5. Analysis of IDS and BFS on Finding Mate-in-N Chess Puzzle

    The minimax algorithm is crucial in game theory and AI for solving chess puzzles like Mate-in-N, which involves achieving checkmate in a specified number of moves. This study analyzes the complexity of Iterative Deepening Search (IDS) and Breadth-First Search (BFS) within the minimax framework, enhancing understanding of their efficiencies in combinatorial game problems.
